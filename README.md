# Init-os-template

## Ubuntu
For ubuntu:
```
sudo sh ubuntu.sh
```

## Debian

### Docker 

Download
```
curl -OL https://raw.githubusercontent.com/thangdoominh/init-os-template/dev/debian.sh
```

Install
```
sudo sh debian.sh
```
After that run `exit` and try to ssh again

To install Docker Engine, you need the 64-bit version of one of these Debian versions:
- Debian Bookworm 12 (stable)
